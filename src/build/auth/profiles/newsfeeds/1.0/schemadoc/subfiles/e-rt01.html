Refer to ITS Ruby for further information.

Allowed parents

its:rtc and its:ruby

Allowed children

This element may contain text.

This element may contain the following children: abbr, address, annoref, annotation, citation, code, definition, emph (text variant), emph (phrase variant), expansion, ln, name, num, object (text variant), object (phrase variant), quote, ref, its:ruby, s, span (text variant), span (phrase variant), sub, sup, term, time and w

Note that in addition to restrictions presented in the content model above, use of this element must also respect the following requirement:
  • The its:rt element must not contain its:ruby descendants.
Such requirements take precedence over any conflicting statements in the content model or in the lists above of allowed children and parents.
