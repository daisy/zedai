The section element represents a major structural division of a verse , such as a canto.

Usage Example

<verse>
    <section>
        <ln>O, never say that I was false of heart,</ln>
        <ln>…</ln>
    </section>
</verse>

Allowed parents

section and verse

Allowed children

This element must not contain text.

This element may contain the following children: annotation, h, hd, ln, lngroup, note, pagebreak and section
