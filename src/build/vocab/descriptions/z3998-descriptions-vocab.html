About this vocabulary

This RDF vocabulary defines a set of terms relating to accessible descriptions.

You are currently reading the normative W3C XHTML+RDFa expression of this vocabulary. It is also available in informative Notation3 and RDF/XML syntax versions.

Document last modified: 2012-12-19. Refer to the revision history for details.

Description Metadata

Standard terms used for defining metadata about a description.

purpose

The purpose of the description.

descriptionQuality

A quality rating for the description.

Target Readers

targetAge

The target age of the reader.

targetGrade

Target school grade level of the reader.

Description Versioning

currentVersion

A URI pointing to the location where the most current version of the description can be found.

thisVersion

A URI pointing to the location where the currently referenced description can be found.

previousVersion

A URI pointing to the location where the previous version of the description can be found.

alternateVersion

A URI pointing to the location where an alternative version is available.

Repository Access

repository

A URI pointing to a repository of descriptions.

queryConcept

A term relevant to the current description that can be used to query a repository for additional information.
