Z3986-2010 Resource Directory Vocabulary

This vocabulary defines a set of properties for use in the Z39.86-2010 Resource Directory document type.

The base grammar for a Z39.86-2010 Resource Directory instance is W3C XHTML+RDFa, with additional RDF property usage constraints expressed by the normative Z3986-2010 Resource Directory ISO Schematron schema.

maintenance-agency
the maintenance agency
specification-compliance
the specification which the profile or feature conforms to
token-identifier
A canonical string identifier of the profile or feature
normative-schema
the normative schema
normative-prose
Normative prose regarding the use of the profile or schema.
feature
a feature used by the profile
default-vocabulary
the profile's default vocabulary
schema-documentation
the schema documentation
informative-schema
informative schema
default-css
the default cascading stylesheet used for the profile
supporting-software
describes software that supports this profile
examples
examples
module
a module used in this profile
archive
a downloadable archive of all the profile's documents
reference
a human-readable description of the profile's purpose and structure
prefix
a prefix functioning as a symbolical short form for a URI
