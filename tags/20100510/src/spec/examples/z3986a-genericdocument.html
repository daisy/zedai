Resource Directory for the Z39.86-2010 Authoring and Interchange Generic Document Profile, version 0.2

Introduction

The Z39.86-2010 Generic Document Profile can be used to represent a wide range of documents, typically such that are not targeted by the more specific of the Z39.86-2010 profiles.

Examples of generic documents, targeted by this profile, include:

  • Organisation meeting minutes
  • Schedules, proceedings
  • Information leaflets
  • Letters
  • Documents off the web

Being generic, this profile can also be used as a template starting point for creators of more specific Z39.86-2010 Authoring and Interchange profiles.

This profile is maintained by the ANSI/NISO Z39.86 maintenance committee.

Informative Schemata

The following informative schemata are available:

z3986a-genericdocument.single.rng
A single file version of the normative RelaxNG schema.
z3986a-genericdocument.xsd
A W3C XML Schema (XSD) version of the normative schema. This schema contains approximations. It is not guaranteed that instance documents valid to this schema will also be valid to the normative schema. This schema should be used for authoring purposes only; final validation should always be performed against the normative schema.
