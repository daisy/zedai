DAISY 4.0 Periodicals Vocabulary

These metadata items take an xsd:string value unless otherwise specified.

Publication-wide metadata for electronic feed republishing

edition
The edition number of the publication.
string

Article-specific metadata for electronic feed republishing

positionSection
the ordinal value of the section within the publication that the article appears in; e.g. 1.2, where 1 is the position of the parent section in the publication, and 2 is the position of the parent sub section within the particular section.
decimal
positionSequence
The ordinal value for the article within a parent section
integer
kicker
A piece of text that draws attention to the article.
string
positionPage
The page number of the pubblication that the article appears on.
integer
revision
The revision number for the article.
decimal
destEdition
The edition of the publication that the article appears in.
string
