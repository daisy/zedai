Z39.86 Periodicals RDFa Profile

The Periodicals profile defines a standard set of vocabularies for use in Z39.86-AI profiles that target periodicals, newsfeeds and similar document types.

Default Vocabulary

This profile defines the Z39.86-201x Periodicals Vocabulary as its default vocabulary.

All unprefixed CURIEs must resolve to the URI: http://www.daisy.org/z3986/2011/vocab/periodicals/#

Additional Vocabularies

This profile also defines prefixes for the following additional vocabularies:

  • Z39.86-201x Instance Metadata Vocabulary

    z3986 must be used for the URI: http://www.daisy.org/z3986/2011/vocab/decl/#
  • DCMI Metadata Terms in the /terms/ namespace

    dcterms must be used for the URI: http://purl.org/dc/terms/
  • Z39.86-201x Structural Semantics Vocabulary

    struct must be used for the URI: http://www.daisy.org/z3986/2011/vocab/structure/#
