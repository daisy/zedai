Z39.98 Periodicals RDFa Context

The Periodicals context defines a standard set of vocabularies for use in Z39.98-AI profiles that target periodicals, newsfeeds and similar document types.

Default Vocabulary

This context document defines the Z39.98-2012 Periodicals Vocabulary as its default vocabulary.

All unprefixed CURIEs must resolve to the URI: http://www.daisy.org/z3998/2012/vocab/periodicals/#

Additional Vocabularies

This context document also defines prefixes for the following additional vocabularies:

  • Z39.98-2012 Instance Metadata Vocabulary

    z3998 must be used for the URI: http://www.daisy.org/z3998/2012/vocab/decl/#
  • Dublin Core Metadata Element Set, Version 1.1

    dc must be used for the URI: http://purl.org/dc/elements/1.1/
  • DCMI Metadata Terms in the /terms/ namespace

    dcterms must be used for the URI: http://purl.org/dc/terms/
  • Z39.98-2012 Structural Semantics Vocabulary

    struct must be used for the URI: http://www.daisy.org/z3998/2012/vocab/structure/#
  • DIAGRAM Descriptions Vocabulary

    diagram must be used for the URI: http://www.daisy.org/z3998/2012/vocab/descriptions/#
