Z39.86 Default RDFa Context

The Default context defines a standard set of vocabularies for use in Z39.86-AI profiles.

Default Vocabulary

This context document defines the Z39.86-2012 Structural Semantics Vocabulary as its default vocabulary.

All unprefixed CURIEs must resolve to the URI: http://www.daisy.org/z3986/2012/vocab/structure/#

Additional Vocabularies

This context document also defines prefixes for the following additional vocabularies:

  • Z39.86-2012 Instance Metadata Vocabulary

    z3986 must be used for the URI: http://www.daisy.org/z3986/2012/vocab/decl/#
  • Dublin Core Metadata Element Set, Version 1.1

    dc must be used for the URI: http://purl.org/dc/elements/1.1/
  • DCMI Metadata Terms in the /terms/ namespace

    dcterms must be used for the URI: http://purl.org/dc/terms/
  • DIAGRAM Descriptions Vocabulary

    diagram must be used for the URI: http://www.daisy.org/z3986/2012/vocab/descriptions/#
