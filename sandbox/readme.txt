Get onvdl-20070517.zip from sf and extract its contents (six jars) to this folder.
With java installed, you should now be able to run the ./ZedAI/ZedAILeisure shell scripts.

See also ReadMe.txt in ./ZedAi/ZedAILeisure

If you would like to test validation with relax NG, run ./ZedAI/ZedAILeisure/validateAllRNG.bat.
Some of the documents do not validate due to issues mentioned in ./ZedAi/ZedAILeisure/ReadMe.txt.
