The DAISY Pagenumber module

Definition

A number printed on a page of a document to uniquely identify it. Most contemporary books are paginated consecutively and pages are generally accounted for in the pagination sequence even if a number is not actually printed on the page.

Markup

The intent of using markup to indicate page numbers is to provide direct navigation to a page. It is strongly recommended that pages be individually and uniquely tagged in text books, and recommended that wherever possible, they be included in all DTBs. All pagenum elements must have a unique xml:id attribute.

Page numbers are marked with the pagenum element. There are three types of pages which are distinguished in the markup through use of the page attribute. page="normal" is used to indicate that the content of the number is the standard Arabic numeral used in the body and rear matter of most books. page="front" is used to mark the page numbers used in the front matter of most books (the content of the numbers is very often roman numerals). page="special" is used to indicate variant pagination schemes used in some books, for example hyphenated numbers often used in appendices (A-1, A-2, etc.). page="special" is also used to mark up pages without page numbers, for example, pages with photos which occur between sequentially numbered pages.

Pages that are unnumbered but are a part of the page number sequence, should be marked with the correct page number in the sequence with the appropriate attribute, page="normal", page="front", or page="special".

The pagenum element is not a required element. Electronic documents are often produced without page numbers. Such documents, electronic or print, should be produced without page numbers, exactly as they have been published.

Syntax

 <pagenum page="normal" xml:id="...">...</pagenum>
        <pagenum page="front" xml:id="...">...</pagenum> <pagenum
        page="special" xml:id="...">...</pagenum>

The pagenum element can carry an optional xml:id attribute and a mandatory page attribute.

define: Daisy.page.att.pagefront
attribute: page
value: front
define: Daisy.page.att.pagenormal
attribute: page
value: normal
define: Daisy.page.att.pagespecial
attribute: page
value: special

The content of the pagenum element depends on the type of pagenumber:

  • If the value of the page attribute is front, only roman numerals (either upper or lower case versions of the letters i, v, x, l, c, d and m, in any order) are permitted.
  • If the value of the page attribute is normal, only positive integers are permitted.
  • If the value of the page attribute is special, any set of caharaters is permitted.
