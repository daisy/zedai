DAISY 4.0 Simple Book Profile Role Attribute Values

This profile defines the following values for use in the role attribute as specified in the XHTML Role Attribute Module:

pagenum
pagenum is a separator denoting the break between two contiguous pages.
Related concepts:
XHTML's separator element.
section
section is a generic division of content.
Inherited from:
WAI-ARIA's section role
Related concepts:
DocBook's section element
Sections and subsections in The Chicago Manual of Style
bridgehead
bridgehead is a free-floating heading that is not associated with the hierarchical structure of a document.
Inherited from:
WAI-ARIA's heading role
Related concepts:
XHTML2's h element
Subheads in The Chicago Manual of Style
sidebar
sidebar contains supplemental information, often formatted as an inset.
Inherited from:
WAI-ARIA's complementary role
Related concepts:
DocBook's sidebar element
